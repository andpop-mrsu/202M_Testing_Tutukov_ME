Позитивные тесты:
1. Тест с равносторонним треугольником:
   Ввод: (3, 3, 3)
   Ожидаемый результат: "equilateral"

2. Тест с равнобедренным треугольником:
   Ввод: (4, 4, 3)
   Ожидаемый результат: "isosceles"

3. Тест с правильным разносторонним треугольником:
   Ввод: (5, 6, 7)
   Ожидаемый результат: "nonequilateral"

4. Тест с нецелыми сторонами:
   Ввод: (2.5, 2.5, 3.5)
   Ожидаемый результат: "isosceles"

Негативные тесты:
1. Нулевые или отрицательные стороны:
   Ввод: (0, 4, 5)
   Ожидаемый результат: Исключение IncorrectTriangleSides

2. Стороны не образуют треугольник:
   Ввод: (1, 2, 3)
   Ожидаемый результат: Исключение IncorrectTriangleSides

3. Стороны не являются числами:
   Ввод: ("a", 5, 5)
   Ожидаемый результат: Исключение IncorrectTriangleSides

4. Одна из сторон отрицательная:
   Ввод: (5, -3, 4)
   Ожидаемый результат: Исключение IncorrectTriangleSides

5. Одна сторона больше суммы двух других:
   Ввод: (10, 1, 1)
   Ожидаемый результат: Исключение IncorrectTriangleSides

6. Неудачное сочетание с плавающей точкой:
   Ввод: (1.5, 3.5, 5.0)
   Ожидаемый результат: Исключение IncorrectTriangleSides
